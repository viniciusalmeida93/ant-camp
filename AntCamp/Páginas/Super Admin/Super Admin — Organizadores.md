# Super Admin — Organizadores
**Rota:** `/super-admin/organizers`
**Acesso:** Super Admin
**Componente:** `SuperAdminOrganizers`

---

## O que faz
Lista e gerencia todos os organizadores cadastrados na plataforma.

## Funcionalidades
- Visualizar todos os organizadores
- Ver campeonatos de cada organizador
- Desativar organizador
- Convidar novo organizador (via Edge Function `invite-organizer`)
