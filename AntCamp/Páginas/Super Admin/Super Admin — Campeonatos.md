# Super Admin — Campeonatos
**Rota:** `/super-admin/championships`
**Acesso:** Super Admin
**Componente:** `SuperAdminChampionships`

---

## O que faz
Visão administrativa de todos os campeonatos de todos os organizadores.

## Funcionalidades
- Listar todos os campeonatos da plataforma
- Filtrar por organizador, status, data
- Acessar detalhes de qualquer campeonato
- Publicar / despublicar campeonatos
