# ✅ Checklist de Segurança - AntCamp

## Backup do Banco de Dados

□ Script de backup criado (`scripts/backup-database.sh`)
□ Permissão de execução dada (`chmod +x`)
□ Senha do banco configurada no script
□ Primeiro backup executado com sucesso
□ Arquivo `.sql.gz` gerado
□ Backup copiado para Google Drive
□ Backup copiado para pendrive
□ Adicionado ao `.gitignore`

## Documentação

□ `CONFIGURACOES_CRITICAS.md` criado
□ Todas as credenciais preenchidas:
  □ Supabase (Project ID, Keys, Password)
  □ Asaas (API Key, Wallet ID)
  □ Resend (API Key, Email)
  □ Domínio e DNS
□ Salvo no gerenciador de senhas
□ Backup criptografado no Google Drive
□ Cópia física em pendrive
□ Adicionado ao `.gitignore`

## Secrets e Credenciais

□ Todos os secrets exportados do Supabase
□ Salvos no gerenciador de senhas (1Password/Bitwarden)
□ Backup criptografado criado
□ Arquivo original deletado (segurança)
□ Testado recuperação de um secret

## Git e Segurança

□ `.gitignore` atualizado
□ Nenhum arquivo sensível commitado
□ Histórico do Git verificado
□ Branch principal protegida

## Testes

□ Backup testado (executar script)
□ Restauração testada (opcional)
□ Acesso a todas as contas verificado
□ Links de dashboards salvos

## Procedimentos de Emergência

□ Sabe como restaurar backup
□ Sabe como recuperar secrets
□ Contatos de suporte salvos
□ Plano B documentado
